# Release Notes

## 1.0.1

- Fixed extension activation in Nova.
- Added Extension Library artwork and publishing documentation.
- Added an npm non-affiliation disclaimer and streamlined the public documentation.

## 1.0.0

- Added npm package details when hovering over dependencies in `package.json`.
- Added code actions and an editor command for updating dependencies to the latest release.
- Preserved leading `^` and `~` version operators when updating.
- Added support for dependencies, development dependencies, peer dependencies, and optional dependencies.
