# Package Peek for Nova

Package Peek shows current npm package details and the latest release when hovering over dependencies in `package.json`. It also provides a code action and editor command for updating a dependency to the latest version.

See the [extension documentation](./Package%20Peek.novaextension/README.md) for installation, features, requirements, and privacy details.
